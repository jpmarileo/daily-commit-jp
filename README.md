## 📌 ¿Qué es este proyecto?

`daily-commit-jp` es un **GitHub Action** que realiza commits automáticos diarios en este mismo repositorio para mantener activa tu **GitHub streak** (racha de contribuciones).

Ideal para quienes, como yo, estamos volviendo fuerte a la programación después de un tiempo trabajando en otras áreas y queremos mostrar actividad constante mientras aplicamos a trabajos remotos en desarrollo de software.

---

## ✨ Características

- Commit automático todos los días
- Configurado con **GitHub Actions** (workflow en `.github/workflows/daily-commit.yml`)
- Fácil de personalizar (mensaje, archivos, horario)
- Ligero y 100% gratuito
- Pensado para programadores que quieren mantener el hábito y la visibilidad en GitHub

## 🚀 Cómo funciona

1. El workflow se ejecuta automáticamente cada día gracias a `schedule` en GitHub Actions.
2. Crea o actualiza un archivo (por ejemplo, un log de fecha).
3. Hace commit y push con un mensaje claro.
4. ¡Tu contribución verde aparece en tu gráfico de GitHub!

---

## 🔧 Cómo usarlo en tu propio repositorio

1. Forkea este repo o copia el contenido de `.github/workflows/daily-commit.yml`
2. Pégalo en tu propio repositorio en la misma ruta.
3. Activa los workflows (normalmente se activan solos).
4. ¡Listo! En 24 horas deberías ver el primer commit automático.

Quieres personalizar el mensaje o el archivo que se modifica? Edita el archivo `daily-commit.yml` y ajusta las variables.
